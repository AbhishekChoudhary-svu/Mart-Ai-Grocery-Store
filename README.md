# Mart AI – AI-Powered Grocery Store with Admin Panel

Mart AI is a modern, intelligent grocery shopping platform designed to provide a seamless experience for both customers and administrators. The solution integrates AI-powered features to enhance product discovery and shopping convenience while offering a comprehensive admin panel to efficiently manage products, orders, and users.

## Live Demo

- **Client Website:** [https://mart-ai-grocery-store-main.vercel.app](https://mart-ai-grocery-store-main.vercel.app)
- **Admin Panel:** Access available via project repo or by contacting the owner.

## Features

- Browse, search, and filter products with ease.
- Intuitive shopping cart and secure checkout process.
- Recipe Kit feature that allows adding all recipe ingredients to the cart with a single click.
- Robust admin panel for management of products, categories, users, and orders.
- Secure user authentication implemented via JWT and Firebase.
- Cloud-hosted media management using Cloudinary for optimized image delivery.
- Comprehensive order management functionality within the admin panel.

## Project Structure

- **/client** – User-facing frontend application built with React.
- **/admin** – Administrative dashboard for store operations.
- **/server** – Backend API developed with Node.js and Express.

## Setup and Usage

1. Clone the repository:
   ```
   git clone https://github.com/your-username/mart-ai-grocery-store.git
   ```

2. Install dependencies for each module:
   - Client:
     ```
     cd client
     npm install
     ```
   - Admin:
     ```
     cd ../admin
     npm install
     ```
   - Server:
     ```
     cd ../server
     npm install
     ```

3. Configure environment variables for each module as required.

4. Run the applications:
   - Client:
     ```
     npm start
     ```
   - Admin:
     ```
     npm start
     ```
   - Server:
     ```
     npm run dev
     ```

5. Access the client application at `http://localhost:3000` (or deployed URL).

6. Access the admin panel at `http://localhost:3001` (or deployed URL).
